
# vscode-win-keymap-on-mac
vscode-win-keymap-on-mac

### usage:
`mvn package`

```bash
cd win-keymap-on-mac
npm install @vscode/vsce
node_modules/.bin/vsce --version
node_modules/.bin/vsce package
```
